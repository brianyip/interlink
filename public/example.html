Interlink JavaScript Snippet Example

This page demonstrates how the Interlink JavaScript snippet works to replace card placeholders with compliant card references.

Basic Card References

Here are some example card references that will be replaced:

  • The {{Card:ChaseSapphirePreferred}} offers great travel benefits.
  • Consider the {{Card:AmexGold}} for dining rewards.
  • The {{Card:CapitalOneSavor}} is excellent for cash back.

Custom Display Text

You can also use custom display text:

  • Check out this {{Card:ChaseSapphirePreferred|custom=amazing travel card}} for your next trip.
  • The {{Card:AmexGold|custom=premium dining card}} offers 4X points on restaurants.

Fallback Behavior

If a card key doesn't exist, it will fall back gracefully:

  • This {{Card:NonExistentCard}} will remain unchanged.
  • Custom text fallback: {{Card:AnotherNonExistent|custom=custom text here}}

Usage Instructions

  1. Include the Interlink script with your user ID: 
    <script src="https://your-domain.vercel.app/js/interlink.min.js" data-user-id="YOUR_USER_ID" defer></script>
  2. Use placeholders in your content: {{Card:YourCardKey}}
  3. Optionally add custom text: {{Card:YourCardKey|custom=Your Custom Text}}

Configuration Options

  • data-user-id: Your unique user ID (required)
  • data-replace-all: Set to "false" to replace only first occurrence (default: true)
