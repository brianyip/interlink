Interlink JavaScript Snippet Example

This page demonstrates how the Interlink JavaScript snippet works to replace link placeholders with dynamic content.

Basic Link References

Here are some example link references that will be replaced:

  • The {{chasesapphirepreferred}} offers great travel benefits.
  • Consider the {{amexgold}} for dining rewards.
  • The {{capitalonesavor}} is excellent for cash back.

Custom Display Text

You can also use custom display text:

  • Check out this {{chasesapphirepreferred|custom=amazing travel card}} for your next trip.
  • The {{amexgold|custom=premium dining card}} offers 4X points on restaurants.

Fallback Behavior

If a link key doesn't exist, it will fall back gracefully:

  • This {{nonexistentlink}} will remain unchanged.
  • Custom text fallback: {{anothernonexistent|custom=custom text here}}

Usage Instructions

  1. Include the Interlink script with your user ID: 
    <script src="https://YOUR-INTERLINK-DOMAIN.com/js/interlink.min.js" data-user-id="YOUR_USER_ID" defer></script>

    Note: The script automatically detects the domain from the script source URL, so it works with any deployment domain.

  2. Use placeholders in your content: {{yourlinkkey}}
  3. Optionally add custom text: {{yourlinkkey|custom=Your Custom Text}}

Configuration Options

  • data-user-id: Your unique user ID (required)
  • data-replace-all: Set to "false" to replace only first occurrence (default: true)
